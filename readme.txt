== Description ==

 ⁃ ArvanCloud’s automatic caching helps you easily remove your Wordpress website and show new content to your website audience, including posts or comments at any time.

 ⁃  ArvanCloud CDN is the best solution for Wordpress websites. No matter what hosting service you use for your website, ArvanCloud is compatible with all hosting providers and owning several international pop sites, it performs efficiently for users all around the world.

 ⁃ ArvanCloud CDN provides you with the following features :

 • Minimizing the size of your website’s content
 • SEO optimization
 • DDoS Mitigation
 • Saving Traffic
 • Access to your website even during down time
 • Analytics and statistics

== Installation ==

1. Upload arvancloud-cache-cleaner folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Create a ArvanCloud account at https://www.arvancloud.com/register
1. Enter your ArvanCloud Account & API token & API Domain & Domain on the plugin settings page
